## Hardhat

#### Install hardhat

```shell
npm install --save-dev hardhat

```

#### Help

```shell
npx hardhat help

```

#### Deploy

```shell
npx hardhat run scripts/your_directory  --network localhost

```

#### Run local node

```shell
npx hardhat node

```

#### Compile

```shell
npx hardhat compile

```

#### Run test

```shell
npx hardhat test test/your_directory

```

#### Hardhat console

```shell
npx hardhat console --network localhost

```
